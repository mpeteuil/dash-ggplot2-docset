Given a character vector, create a set of identity mappings

Usage

aes_all(vars)

Arguments

vars
vector of variable names

Description

Given a character vector, create a set of identity mappings

Examples

aes_all(names(mtcars))



* mpg  -> mpg
* cyl  -> cyl
* disp -> disp
* hp   -> hp
* drat -> drat
* wt   -> wt
* qsec -> qsec
* vs   -> vs
* am   -> am
* gear -> gear
* carb -> carb



aes_all(c("x", "y", "col", "pch"))



* x      -> x
* y      -> y
* colour -> col
* shape  -> pch
